# Wellness Platform Monorepo

This monorepo contains three projects for the Wellness platform.

## Projects

| Project | Tech Stack | Directory |
|---------|-----------|-----------|
| **wellness-core-backend** | NestJS + PostgreSQL | `wellness-core-backend/` |
| **wellness-admin** | React + Vite | `wellness-admin/` |
| **wellness-app** | Flutter | `wellness-app/` |

## Getting Started

### Backend
```bash
npm run dev:backend
```

### Admin Panel
```bash
npm run dev:admin
```

### Mobile App
```bash
cd wellness-app
flutter run
```
